友好提示:

1.  更换珊瑚缸照片:

点击APP界面上的珊瑚缸图标3秒，根据提示进行照片更换。.

2.  LED灯按先后顺序连接到家庭网络/办公网络:

如果您有多个G4水族灯，请先给一个灯上电，通过APP连接到家庭网络，连接成功后再给第二个灯上电连接。 否则有可能会因为对APP不熟悉导致将一个灯连接到另外一个灯。

3.  如何区分各个LED灯:

如果您有多个灯，请在在线设备列表中选取一个灯，长按3秒，对应的灯将闪三次，这样您就可以区分哪个IP地址对应哪个灯。

该功能在新增群组的界面中同样有效。

4.  如果区分各个组的灯:

如果您的灯分了几个组，可以长按组号3秒，该组内的所有LED灯将闪3次。

3). 点击图标“   ”:

8). APP将目标LED灯的WIFI重新配置，请等大约一分钟，在主界面点击“搜索”，新增加的LED灯将出现在在线设备列表中：

 

2. 手机和LED灯WIFI直连:

如果您的LED灯周围并没有家庭网络或者办公网络，可以采用手机和LED灯WIFI直连。

1). 打开手机的设置，并打开WIFI设置；

2). 找到名字为“CTLite-xxxx”的网络，输入密码“12345678”并点击链接；

3). 运行APP就可以开始控制您的LED灯。

3. 删除LED灯:

1). 软件删除：

A．点击 “删除设备”; B. 选择一个或者多个目标LED灯; C. 点击 “确认”。

2). 硬件复位删除：

长按灯具天线边上的红色按钮15秒，灯具WIFI模块将自动复位并恢复出厂设置。

WIFI模块成功复位后，手机可以搜索到该灯具的WIFI信号。

第四部分: LED灯的控制：

点击某组号右端的“控制”进入改组灯的控制界面.

1.  “自动”工作模式:

1). 如何选择时间节点:

请用手指在图表上左右滑动来选择时间节点；

2). 增加时间节点:

- 选择好时间节点后，点击 “+”:

-          设置好该时间节点每个LED通道的发光百分比;

-          点击 “确认”，该时间节点添加成功。

2). 删除时间节点:

-  点击 “〈”  或者  “〉” 来选择已经设置好的时间节点；

-  点击 “-”删除该时间节点；

3). 编辑时间节点:

-  请用手指在图表上将垂直时间轴移动到存在的时间节点或者点击 “〈” /  “〉” 选择要编辑的时间节点；

-  点击 “编辑”；

-  重新设置该时间节点每个LED通道的发光百分比；

-  点击 “确认”，该时间节点编辑成功。

4). 恢复出厂设置:

点击“出厂设置”将曲线恢复到出厂设置。

5). 设置预览:

点击 “    ” 来预览曲线设置。

2.  “手动”工作模式:

-          点击“手动”进入手动工作模式；

-          调节每个通道的发光比例，该组的LED灯各通道一天都按这个设置去发光。

第五部分：效果

1.  月光模式:

1). 点击月光模式区域来开启或关闭月光工作模式；

2). 选择上月的月圆日期，每天的月光模式的发光比例略有不同，按照一个月的月亮周期自动运行；

3). 请设置每天的月光模式开始时间和结束时间；

2.  闪电模式:

1). 点击闪电区域来开启或关闭闪电功能;

2). 请设置每天的闪电开始时间和结束时间，一共有三个时间段可以设置，并带有启用和不启用的选项；

3.  珊瑚适应:

1). 点击珊瑚适应区域来开启或关闭珊瑚适应功能;

2). 该功能开启后，LED灯将按当前设置的50%的比例来发光，之后每天亮度慢慢增加，20天后亮度将大目前设置数值的100%，完成珊瑚适应。
